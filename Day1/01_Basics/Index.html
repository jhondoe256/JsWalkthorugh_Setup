JavaScript Basics!

  1. Variables

    • Variable  : We use variables to temporarily store data in a computers memory. the 'memory' has its own 'address' in which the data is stored. We as developers have the ability to give this 'memory' a name (Variable name) which allows us to read/use the data when nessessary.

    • Think of a variable being a 'box', whats put inside the box is the value that is assigned to the variable (the data). and the label that we put on the box is the variable name

  2. What kind of values can we assign to variables?

      Variables are based on Two Categories of types

        Value Types (Primatives)

      • String
      • Number
      • bool
      • undefined
      • null

        Reference Types

      • Object

        • Objects:   is like an object in 'real-life'. An example would be a Customer. Customer (object) can have properties like:
          1. firstName
          2. lastName
          3. Address
          4. emailAddress
      • Array:  
        Arrays are a collection of items.
        You may have to work with an entire list of objects.
        List of items on your grocery list.
        A collection of grades in a gradebook.
        • Function:
        Function is a set of statements that perform a task or calculates a value.

  3. JavaScript is a Dynamic language

    • The value of the variable can change during runtime!
